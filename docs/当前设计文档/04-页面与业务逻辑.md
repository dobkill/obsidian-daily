# 04. 页面与业务逻辑

## 任务总览

任务总览展示全库层面的统计：

1. 活跃度热力图读取过去 12 个月实例完成情况。
2. 最近 30 天趋势展示每日任务总数与完成数。
3. 周任务概览按日期列展示本周任务实例，支持新增任务、勾选完成、编辑单次实例。

## 今日任务页

今日任务页读取 `getTasksForDate(today)`，并将组合任务和挂入组合任务的真实子任务汇总展示。

操作：

1. 普通任务勾选调用 `updateTaskOccurrenceCompletion()`。
2. 组合任务轻量子项勾选调用 `updateTaskOccurrenceSubtaskCompletion()`。
3. 编辑任务打开 `TaskModal`。
4. 导出今日任务调用 `exportTasksAsMinimalCompletionText()`，只复制极简今日完成格式。

## 项目进度页

项目进度页按 `ProgressPage` 顺序展示项目。每个项目有四个视图：

1. 表格：展示任务主字段，支持新增、编辑、删除、批量导入和项目 Markdown 导出。
2. 看板：按 `status` 和 `viewState.board` 展示任务。
3. 甘特图：按日期、时间、`viewState.gantt` 展示任务。
4. 思维导图：按 `viewState.mindmap` 和 `mindmapComments` 展示任务节点与评语节点。

“导出全部记录”位于项目进度页标题区，复制数据迁移 JSON。

## 快速记录页

快速记录页有四个目标：

1. 写日记：写入每日文件或汇总文件。
2. 追加笔记：追加到任意 Markdown 文件。
3. 创建任务：解析数据迁移 JSON、今日完成极简 Markdown、新任务计划复杂 Markdown。
4. 补充导图：把内容追加到任务或评语正文，或逐行创建评语子节点。

## 设置页

设置页顶部展示三种创建格式。每种格式都有说明、可编辑示例文本框和复制按钮。文本框内容不持久化，重新打开设置页恢复默认示例。
